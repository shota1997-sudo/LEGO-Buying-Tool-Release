🧱 Shotval Foundry: LEGO Buying Tool (v1.0.0)

A powerful procurement optimization tool designed specifically for LEGO AFOLs and MOC (My Own Creation) designers.

📥 How to Download?

Please click on the [**Releases**](https://github.com/shota1997-sudo/LEGO-Buying-Tool-Release/releases/latest) section on the right side of this page and download the executable file corresponding to your operating system.

📁 Files Included

*   **LEGO_Tool_Win.exe**: Executable for Windows users.
*   **LEGO_Tool_Mac**: Executable for macOS users.
*   **buy_parts_example.xlsx**: Parts list template (Please follow this format strictly).

🛠️ Quick Start

1.  **Download & Extract**: Download the `.zip` file and extract the contents to a folder.
2.  **Prepare Data**: Open `buy_parts_example.xlsx` and fill in the Part IDs and quantities you wish to purchase.
3.  **Run the Tool**: Execute the program. The tool will automatically optimize the procurement logic and sourcing for you.

⚠️ Notes for macOS Users

Since this software is **not notarized** by Apple, you may encounter a "Developer Cannot be Verified" warning upon launching.

To bypass this:
1.  Right-click the `LEGO_Tool_Mac` file and select **Open**.
2.  Click Open again in the pop-up confirmation dialog.

If the program fails to launch:
You may need to manually grant execution permissions. Open your **Terminal**, navigate to the folder, and run:
`chmod +x LEGO_Tool_Mac`
